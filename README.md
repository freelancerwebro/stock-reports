# Stock Reports

Laravel app for parsing and displaying data from RapidAPI. 

## Installation
```
./vendor/bin/sail up
```

## Migrations
```
php artisan migrate
```

## Seeding
```
php artisan db:seed CompanySeeder
```

## Usage
```
http://localhost
```

## Run tests
```
php artisan test
```
